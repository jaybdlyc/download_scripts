# Weibo Album Bulk Downloader UserScript

这是一个油猴脚本，用于在微博用户相册页面上一键批量下载全部图片和 LivePhoto 视频。

## 功能概述

- 支持在用户微博相册页（`tabtype=album`）批量抓取并下载所有媒体文件。
- 可设置下载间隔、顺序等参数
- 在最新版本中，脚本会先遍历所有媒体列表以预计算时间线（年月），并在下载阶段直接使用这些数据，避免重复调用微博接口，提高效率。

## 使用方法

1. 安装油猴（Tampermonkey、Greasy Fork 等支持 UserScript 的插件）。
2. 将脚本 `weibo_album_bulk_download.user.js` 添加到油猴。
3. 打开微博用户主页并切换到相册标签，页面右下角会出现控制按钮。
4. 点击“开始下载”按钮执行批量抓取并下载。
5. 使用“停止”按钮随时终止当前任务；“暂停/继续”由主按钮控制。
